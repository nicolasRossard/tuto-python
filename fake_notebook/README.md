# Debugging
* directly with [Jupyterlab](https://jupyterlab.readthedocs.io/en/stable/user/debugger.html) using Conda
* With specific lib: IPython.core.debugger
```python
from IPython.core.debugger import set_trace
def bad_function(var):
    set_trace()
    return var + 0
bad_function("Mike")
```
